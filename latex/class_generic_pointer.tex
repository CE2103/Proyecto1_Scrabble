\doxysubsection{Descripción detallada}
\subsubsection*{template$<$typename Value\+Type, typename Allocator = Crt\+Allocator$>$\newline
class Generic\+Pointer$<$ Value\+Type, Allocator $>$}

Represents a J\+S\+ON Pointer. Use Pointer for \mbox{\hyperlink{struct_u_t_f8}{U\+T\+F8}} encoding and default allocator. 

This class implements R\+FC 6901 \char`\"{}\+Java\+Script Object Notation (\+J\+S\+O\+N) Pointer\char`\"{} (\href{https://tools.ietf.org/html/rfc6901}{\texttt{ https\+://tools.\+ietf.\+org/html/rfc6901}}).

A J\+S\+ON pointer is for identifying a specific value in a J\+S\+ON document (\mbox{\hyperlink{class_generic_document}{Generic\+Document}}). It can simplify coding of D\+OM tree manipulation, because it can access multiple-\/level depth of D\+OM tree with single A\+PI call.

After it parses a string representation (e.\+g. \char`\"{}/foo/0\char`\"{} or U\+RI fragment representation (e.\+g. \char`\"{}\#/foo/0\char`\"{}) into its internal representation (tokens), it can be used to resolve a specific value in multiple documents, or sub-\/tree of documents.

Contrary to \mbox{\hyperlink{class_generic_value}{Generic\+Value}}, Pointer can be copy constructed and copy assigned. Apart from assignment, a Pointer cannot be modified after construction.

Although Pointer is very convenient, please aware that constructing Pointer involves parsing and dynamic memory allocation. A special constructor with user-\/ supplied tokens eliminates these.

\mbox{\hyperlink{class_generic_pointer}{Generic\+Pointer}} depends on \mbox{\hyperlink{class_generic_document}{Generic\+Document}} and \mbox{\hyperlink{class_generic_value}{Generic\+Value}}.


\begin{DoxyTemplParams}{Parámetros del template}
{\em Value\+Type} & The value type of the D\+OM tree. E.\+g. \mbox{\hyperlink{class_generic_value}{Generic\+Value}}$<$U\+T\+F8$<$$>$ $>$ \\
\hline
{\em Allocator} & The allocator type for allocating memory for internal representation.\\
\hline
\end{DoxyTemplParams}
\begin{DoxyNote}{Nota}
\mbox{\hyperlink{class_generic_pointer}{Generic\+Pointer}} uses same encoding of Value\+Type. However, Allocator of \mbox{\hyperlink{class_generic_pointer}{Generic\+Pointer}} is independent of Allocator of Value. 
\end{DoxyNote}
