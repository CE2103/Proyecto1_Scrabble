class Generic\+Schema\+Document$<$ Value\+T, Allocator $>$}

J\+S\+ON schema document. 

A J\+S\+ON schema document is a compiled version of a J\+S\+ON schema. It is basically a tree of \mbox{\hyperlink{classinternal_1_1_schema}{internal\+::\+Schema}}.

\begin{DoxyNote}{Nota}
This is an immutable class (i.\+e. its instance cannot be modified after construction). 
\end{DoxyNote}

\hline
