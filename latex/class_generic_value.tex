class Generic\+Value$<$ Encoding, Allocator $>$}

Represents a J\+S\+ON value. Use Value for \mbox{\hyperlink{struct_u_t_f8}{U\+T\+F8}} encoding and default allocator. 

A J\+S\+ON value can be one of 7 types. This class is a variant type supporting these types.

Use the Value if \mbox{\hyperlink{struct_u_t_f8}{U\+T\+F8}} and default allocator
