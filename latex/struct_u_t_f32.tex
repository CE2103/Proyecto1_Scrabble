\href{http://en.wikipedia.org/wiki/UTF-32}{\texttt{ http\+://en.\+wikipedia.\+org/wiki/\+U\+T\+F-\/32}} 
\begin{DoxyTemplParams}{Parámetros del template}
{\em Char\+Type} & Type for storing 32-\/bit U\+T\+F-\/32 data. Default is unsigned. C++11 may use char32\+\_\+t instead. \\
\hline
\end{DoxyTemplParams}
\begin{DoxyNote}{Nota}
implements Encoding concept

For in-\/memory access, no need to concern endianness. The code units and code points are represented by C\+PU\textquotesingle{}s endianness. For streaming, use \mbox{\hyperlink{struct_u_t_f32_l_e}{U\+T\+F32\+LE}} and \mbox{\hyperlink{struct_u_t_f32_b_e}{U\+T\+F32\+BE}}, which handle endianness.
